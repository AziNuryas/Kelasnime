# KelasNime

Nonton anime sub Indo gratis, streaming dan download anime terlengkap.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Bahasa**: TypeScript
- **Styling**: Tailwind CSS
- **Data**: Sanka Vollerei API
- **Icons**: Lucide React

## Development

```bash
npm install
npm run dev
```

## Struktur

```
app/          → Pages & API routes
components/   → Reusable UI components
lib/          → Helpers, constants, API fetch
types/        → TypeScript interfaces
```
